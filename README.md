# PyCraft

## Overview
This is a simple voxel game created using the Ursina engine.

## Prerequisites
* Python 3.x
    * Ursina

## Installation
1. Clone the repository:
```bash
git clone https://github.com/ecnivs/PyCraft.git
```
2. Install dependencies:
```bash
pip install ursina
```
3. Run the Software:
```bash
python main.py
```

## Controls
* `W`: Move forward
* `S`: Move back
* `A`: Move left
* `D`: Move right
* `Left Click`: Remove block
* `Right Click`: Place block
* `1-4`: Change block
